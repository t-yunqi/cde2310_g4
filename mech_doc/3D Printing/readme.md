# 3D Printing

The following files are to be printed with layer lines **parallel** to press fit extrusion direction:

  - Servo 1
  - Servo 2
  - Cam Mount High
  - Cam Mount Low

Suggested Print Settings:
  -3 Walls
  -15% infill
  -Adaptive Cubic infill
  - 45% Tree Support Angle (if supports needed)
  - 5mm brim (optional)
